Create

  • You need a web browser to display htmlwidgets (which rely on HTML, JS, CSS), so you can use Rmarkdown to create Presentations with ioslides.

  • Put the rendered htmlwidgets_save.html (and rendered htmlwidgets_save.html) in a new docs/ folder, turn on Github Pages using docs as Source in your repository's Settings (requires admin privileges), and you'll be able to publicly view it at https://mhepner90.github.io/RVC/htmlwidget_save.html.

Interactive

  • CSS font styles might clash between standalone html and embedded in presentation, which is fixable with some work

Static

  • You might need to crop and resize image in image editor (eg Preview on Mac) to fit on slide
